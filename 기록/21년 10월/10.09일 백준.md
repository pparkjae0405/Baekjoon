02. 10718
```
public class Main {
    public static void main(String args[]){
        System.out.println("강한친구 대한육군");
        System.out.println("강한친구 대한육군");
    }
}
```
풀이
```
주어진 문장을 두번 출력하는 문제
for문은 오히려 시간이 더 걸리므로 System.out.println()문을 두 번 사용
```
