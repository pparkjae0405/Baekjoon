# 10/11 백준 코딩테스트

03. 10171
```
public class Main {
    public static void main(String args[]){
        System.out.println("\\    /\\");
        System.out.println(" )  ( ')");
        System.out.println("(  /  )");
        System.out.println(" \\(__)|");
    }
}
```
풀이
```
주어진 고양이 모양을 출력하는 문제
\ 하나를 출력하려면 \\를 입력해야 함
```
04. 10172
```
public class Main {
    public static void main(String args[]){
        System.out.println("|\\_/|");
        System.out.println("|q p|   /}");
        System.out.println("( 0 )\"\"\"\\");
        System.out.println("|\"^\"`    |");
        System.out.println("||_/=\\\\__|");
    }
}
```
풀이
```
주어진 개 모양을 출력하는 문제
위와 같이 \ 는 \\로, "는 \"로 입력해야 함
```
