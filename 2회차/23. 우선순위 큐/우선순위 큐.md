# 우선순위 큐

```
- 개념
각 요소들이 우선순위를 갖고있고, 우선순위가 높은 요소가 먼저 제공되는 자료구조

- 사용법
PriorityQueue<자료형> a = new PriorityQueue<자료형>();
Collections.reverseOrder()로 내림차순 가능

- Comparator
객체를 비교할 수 있도록 만드는데
compare(Object left, Object right)의 리턴값이
left right 순으로 와야 할 경우 -1 리턴
right left 순으로 와야 할 경우 1 리턴
left와 right가 같을 경우 0을 리턴한다
```
