# 집합과 맵

```
- 집합
중복되지 않는 원소들의 모음을 만들 수 있는 자료구조

- 자료구조
해시 셋(HashSet)
트리 셋(TreeSet) : 이진 탐색 트리의 형태로 저장
연결 해시 셋(LinkedHashSet)

- 사용법
HashSet<자료형> a = new HashSet<자료형>();
TreeSet<자료형> a = new TreeSet<자료형>();
LinkedHashSet<자료형> a = new LinkedHashSet<자료형>();
로 선언하고 a.명령어()로 사용

- 맵
Key와 Value 한 쌍으로 이루어진 자료형, Value는 중복될 수 있지만 Key는 고유한 값을 가져야 함

- 자료구조
해시 맵(HashMap)
트리 맵(TreeMap) : 이진 탐색 트리의 형태로 저장, 입력된 key의 소트 순으로 데이터가 출력되는 특징
연결 해시 맵(LinkedHashMap) : 입력된 순서대로 데이터가 출력되는 특징

- 사용법
HashMap<자료형, 자료형> a = new HashMap<자료형, 자료형>();
```
