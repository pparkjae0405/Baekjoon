# 문자열

```
- 입력한 문자열의 아스키코드 값 출력
int N = br.read();
System.out.println(N);

- 문자열 하나씩 자르기
문자열.charAt(i)

- 조건에 따라 문자열 자르기
StringTokenizer st = new StringTokenizer(br.readLine(), "조건");

- 문자열 소문자 대문자 변환
문자열.toLowerCase();
문자열.toUpperCase();

- 문자열 더하여 출력
StringBuilder sb = new StringBuilder();
sb.append("출력내용");
System.out.println(sb);
```
