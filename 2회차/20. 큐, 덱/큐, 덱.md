# 큐, 덱

```
- 큐 개념
먼저 들어온 데이터가 먼저 나가는 FIFO(First In First Out, 선입 선출) 구조

- 사용법
큐는 첫 번째 저장된 데이터를 삭제하므로, ArrayList와 같은 배열 구조의 자료구조를 사용하게 되면 빈 공간을 채우기 위해 데이터의 복사가 발생하므로 데이터의 추가/삭제가 쉬운 LinkedList를 사용
QueueStack<자료형> a = new LinkedList<자료형>();
로 선언하고 a.명령어()로 사용

- 덱 개념
큐의 양쪽에 데이터를 넣고 뺄 수 있는 형태의 자료구조

- 사용법
Deque<자료형> a = new LinkedList<자료형>();
로 선언하고 a.명령어()로 사용
```
