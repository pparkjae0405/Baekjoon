# 입출력과 사칙연산

```
- 입력 (Scanner)
import java.util.Scanner;
Scanner sc = new Scanner(System.in); 으로 선언, sc. 으로 사용

- 입력 (BufferedReader)
import java.io.*;
main 함수에 throws IOException 추가
BufferedReader br = new BufferedReader(new InputStreamReader(System.in)); 으로 선언, br. 으로 사용

- 출력
System.out.print("출력내용");

- 출력 및 줄바꿈
System.out.println("출력내용");

- 출력 (BufferedWriter)
import java.io.*;
main 함수에 throws IOException 추가
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out)); 으로 선언, bw. 으로 사용

- String -> int 형변환
Integer.parseInt("123");
```
