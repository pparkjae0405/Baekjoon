# 이분 탐색

```
- 개념
정렬된 배열 또는 시르트에 적합한 고속 탐색 방법

- 사용법
1. 탐색의 대상이 되는 자료를 정렬시킨다.
2. low와 high값의 중간값 mid값은 (low + high) /2이다.
3. 중간값과 구하고자 하는 key값을 비교한다.
   key > mid라면 low를 mid+1로 만든다(왼쪽 반을 버림)
   key < mid라면 high를 mid-1로 만든다(오른쪽 반을 버림)
   key == mid라면 구하고자 하는 값이므로 중간값을 리턴한다.
4. low > high가 될 때까지 1~3번을 반복하여 값을 찾는다.
```
