# 재귀

```
- 개념
특정 함수 내에서 자기 자신을 다시 호출하여 문제를 해결해나가는 함수
문제를 해결하기 위해 원래 범위에서 더 작은 범위의 하위 문제를 먼저 해결함으로써 원래 문제를 해결해 나가는 방식

- 사용법
1. 메인 함수에서 함수 호출
2. 끝날 지점의 return값 지정
3. 그 외에는 조건에 맞게 자기 자신을 호출

- 예시
팩토리얼 함수
예외 1 : 0이 들어올 경우 1 리턴
끝날 지점 : N이 1이하가 될 경우 N 리턴
조건 : N * (N-1)

하노이 탑
끝날 지점 : n이 1이라면 a -> c 이동
조건 : n-1개의 원판을 a -> b로 이동 후(n-1, a, c, b) 
        남은 1개 원판을 a -> c로 이동,
        n-1개의 원판을 b -> c로 이동(n-1, b, a, c)
```
